# Media Date Renamer

A lightweight Windows utility for renaming media files based on their creation or metadata dates.

**Author:** Z

---

## Overview

Media Date Renamer helps organize photos and videos by renaming files according to their actual creation date.

This is especially useful when:
- Files have inconsistent naming conventions
- Media has been copied between devices
- Chronological sorting is important

---

## Features

### 🕒 Date-Based Renaming
- Rename files using creation date or metadata date
- Supports common media formats (images and videos)

### 📂 Batch Processing
- Process entire folders at once
- Handles large collections efficiently

### 🧠 Smart Date Detection
- Uses file metadata when available (e.g., EXIF)
- Falls back to file system timestamps when needed

### ⚡ Fast & Lightweight
- Minimal system requirements
- Quick processing with low overhead

## Quick Start

1. Download the latest release
2. Extract the ZIP file
3. Run `MediaDateRenamer.exe`
4. Select a folder containing media files
5. Choose renaming options
6. Start the renaming process
